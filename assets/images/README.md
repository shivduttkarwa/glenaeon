# Images Assets

Place the following images in this folder:

## Required Images

1. **hero-bg.jpg** - Hero section background image (people watching sunset)
   - Recommended size: 1920x1080 or larger
   - Format: JPG for better compression

2. **winter-solstice.jpg** - Event thumbnail for "Coming Up" box
   - Recommended size: 140x180 px
   - Format: JPG or PNG

3. **favicon.png** - Website favicon
   - Recommended size: 32x32 px
   - Format: PNG

## Image Sources from Figma

The hero background and event images can be exported from the Figma design:
- File: Glenaeon-Concordia-Kit
- Export at 2x resolution for retina displays

## Temporary Placeholder

You can use placeholder images from:
- https://placeholder.com
- https://picsum.photos

Example for hero background:
```html
<img src="https://picsum.photos/1920/1080" alt="Hero Background">
```
